# Feel Diary
A NextJS full-stack app to track your mood throughout the years.

Used: 
- Next 15
- Firebase (Authentication and Data Storage)
- Netlify (Site Deployment)
To see the finished project -> [FeelDiary](https://feeldiary.netlify.app)
